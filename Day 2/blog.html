Hướng dẫn xây dựng ứng dụng Todo List với HTML, CSS và JavaScript

Bước 1: Tạo cấu trúc HTML cơ bản

Trước hết, chúng ta cần tạo một khung HTML để hiển thị ứng dụng Todo List. Nó sẽ bao gồm một input để nhập công việc mới, một nút thêm, và một danh sách hiển thị các công việc. 

!DOCTYPE html>
        <html>
        <head>
        <title>Todo List</title>
        </head>
        <body>
        <h1>Todo List</h1>
        <input type="text" id="todo-input" placeholder="Nhập công việc" />
        <button id="add-btn">Thêm</button>
        <ul id="todo-list"></ul>
        </body>
        </html>

Giải thích: Chúng ta có một input với id todo-input để người dùng nhập công việc, một nút add-btn để thêm công việc, và một danh sách ul với id todo-list để hiển thị các công việc.

Hướng dẫn xây dựng ứng dụng Todo List (HTML, CSS, JavaScript)

Trong bài viết này, chúng ta sẽ xây dựng một ứng dụng Todo List đơn giản với các chức năng cơ bản:

  • Thêm công việc mới
  • Hiển thị danh sách công việc
  • Xóa công việc

Ứng dụng sử dụng HTML để tạo cấu trúc, CSS để tạo giao diện, và JavaScript để xử lý logic. Bài viết được trình bày từng bước, mỗi khối code đều có mô tả trước và giải thích sau để bạn dễ theo dõi.

1.Phân tích chức năng

Trước khi code, ta cần xác định:

  • Người dùng nhập nội dung công việc
  • Nhấn nút Add để thêm vào danh sách
  • Mỗi công việc có nút Delete để xóa

Mô tả

4. Xử lý logic bằng JavaScript

Mô tả

JavaScript sẽ:

  • Lắng nghe sự kiện click nút Add
  • Tạo phần tử <li> mới
  • Gắn nút xóa cho từng todo
  • Xóa todo khi nhấn Delete

Code JavaScript (script.js)

    const todoInput = document.getElementById('todoInput');
        const addBtn = document.getElementById('addBtn');
        const todoList = document.getElementById('todoList');
        
        
        addBtn.addEventListener('click', addTodo);
        
        
        function addTodo() {
        const todoText = todoInput.value.trim();
        
        
        if (todoText === '') {
        alert('Vui lòng nhập công việc');
        return;
        }
        
        
        const li = document.createElement('li');
        li.textContent = todoText;
        
        
        const deleteBtn = document.createElement('button');
        deleteBtn.textContent = 'Delete';
        deleteBtn.className = 'delete-btn';
        
        
        deleteBtn.addEventListener('click', () => {
        li.remove();
        });
        
        
        li.appendChild(deleteBtn);
        todoList.appendChild(li);
        
        
        todoInput.value = '';
        }

Giải thích

  • addEventListener: bắt sự kiện click
  • createElement: tạo phần tử HTML bằng JavaScript
  • trim(): loại bỏ khoảng trắng đầu/cuối
  • li.remove(): xóa công việc khỏi danh sách

Bước 3: Viết JavaScript để thêm công việc

Chúng ta sẽ tạo một function để lấy giá trị từ input, tạo một <li> mới và thêm vào danh sách.

<script>
    const addBtn = document.getElementById('add-btn');
    const input = document.getElementById('todo-input');
    const list = document.getElementById('todo-list');

    addBtn.addEventListener('click', function() {
        const todoText = input.value.trim();
        if(todoText !== '') {
            const li = document.createElement('li');
            li.textContent = todoText;
            list.appendChild(li);
            input.value = '';
        }
    });
</script>

Giải thích: Khi người dùng bấm nút thêm, chúng ta lấy giá trị trong input. Nếu không rỗng, tạo một thẻ <li> mới, gán giá trị và thêm vào <ul>. Sau đó xóa input để chuẩn bị nhập công việc tiếp theo.

Bước 4: Thêm nút xóa cho từng công việc

Chúng ta sẽ tạo một nút xóa cho mỗi item để có thể loại bỏ công việc khỏi danh sách.

addBtn.addEventListener('click', function() {
    const todoText = input.value.trim();
    if(todoText !== '') {
        const li = document.createElement('li');
        li.textContent = todoText;

        const deleteBtn = document.createElement('button');
        deleteBtn.textContent = 'Xóa';
        deleteBtn.addEventListener('click', function() {
            list.removeChild(li);
        });

        li.appendChild(deleteBtn);
        list.appendChild(li);
        input.value = '';
    }
});

Giải thích: Mỗi khi thêm công việc, chúng ta tạo thêm một nút 'Xóa' bên trong <li>. Khi bấm vào nút này, công việc đó sẽ bị xóa khỏi danh sách <ul> bằng phương thức removeChild.

Bước 5: Thêm khả năng thêm công việc khi nhấn Enter

Để tiện lợi hơn, chúng ta có thể cho phép người dùng nhấn Enter để thêm công việc.

input.addEventListener('keypress', function(e) {
    if(e.key === 'Enter') {
        addBtn.click();
    }
});

Giải thích: Chúng ta lắng nghe sự kiện keypress trên input. Nếu người dùng nhấn Enter, chúng ta kích hoạt click vào nút thêm, giúp thêm công việc mà không cần bấm chuột.

Kết quả

Với các bước trên, bạn đã có một ứng dụng Todo List cơ bản, có thể:

  • Nhập và thêm công việc mới.
  • Hiển thị danh sách các công việc.
